## installitaion
#### Python 3.7+ is required. Make sure your default python version is >=3.7 by typing python3

## Input File

```bash
%%%G%%%%%%
%--------%
%-%%%%-%-%
%---%--%-%
%-%-%-%%-%
%-----%--%
%%%%%-%%%%
%-%-%-%%-%
%--------%
%%%%%%%S%%
```

* `% -> wall`
* `- -> path`
* `S -> start`
* `G -> end`

## Output File
show a node in path by `@` character
```bash
%%%@%%%%%%
%--@--@--%
%-%%%%-%-%
%---%@@%-%
%-%-%-%%-%
%----@%--%
%%%%%-%%%%
%-%-%-%%-%
%----@-@-%
%%%%%%%@%%
```
